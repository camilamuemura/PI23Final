Sobre o Projeto

Este projeto foi desenvolvido durante a disciplina de Projeto Integrador I do Eixo Computação - Turma 2021 da Univesp.
A ideia surgiu a partir da necessidade de um melhor acompanhamento e facilidade na localização de focos de incêndios e queimadas numa escala nacional. Com base em pesquisa feita junto a equipe do INPE, foi decidido pelo desenvolvimento de um WebGIS em que o usuário possa visualizar focos de incêndio, a fim de fomentar um melhor planejamento do poder público, empresas e comunidade em geral no acompanhamento e resolução de incidentes com queimadas. Foram utilizados para desenvolvimento da plataforma: Postgres para banco de dados, Leaflet para elaboração do mapa, usando HTML, CSS e Javascript, Python como linguagens backend e Django/Geodjango como framework. O controle de versão foi feito através do GitHub. O projeto foi hospedado no Portainer.
